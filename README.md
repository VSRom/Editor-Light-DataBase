№ Editor Light DataBase
Универсальный клиент для управления базами данных с графическим интерфейсом на C++ с использованием Qt.
**Проект находится на ранней стадии разработки.**

## Технологии

    C++17
    Qt 6 (Widgets, Sql)
    CMake (сборка)
    Git (контроль версий)
    SQLite

## Планируемый функционал

Подключение к базам данных:

    Выбор типа СУБД (Oracle, PostgreSQL, MySQL, SQLite, MS Access)
    Ввод адреса, порта, логина и пароля
    Проверка соединения перед подключением
    Сохранение конфигураций с шифрованием паролей
    Менеджер сохранённых подключений

Работа с данными:

    Просмотр списка таблиц базы данных
    Просмотр и редактирование данных в таблицах
    Поиск по значениям в любом столбце
    Добавление, изменение, удаление записей
    Отмена/повтор действий (Ctrl+Z / Ctrl+Y)

Управление структурой БД:

    Создание новых таблиц
    Переименование таблиц
    Удаление таблиц
    Объединение данных нескольких таблиц через SQL-запрос

Дополнительно:

    Локальные заметки (хранятся отдельно от БД)
    Автосохранение состояния при закрытии
    Экспорт данных (в разработке)
